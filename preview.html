PresupuestoPro

Herramienta para presupuestar, generar PDF y acelerar el envío al cliente.
Proyecto real

PresupuestoPro nació para resolver un problema simple pero repetido: perder tiempo armando presupuestos de forma manual, desordenada o poco profesional. El objetivo fue crear una herramienta rápida, clara y lista para usarse en contexto real.

Hoy el proyecto cuenta con una base de casi 100 trabajadores de distintos rubros, lo que lo convierte en el caso más importante de este portfolio: no es un concepto, es un producto pensado para ser usado.
